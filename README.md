# operators
Java operators are special symbols used to perform mathematical, logical, and relational manipulations on variables and values. 
They act as the foundational instructions that direct data processing and control logic execution within a program

Types of Java Operators
Java groups its operators into the following primary categories:
Arithmetic Operators: Used to perform fundamental mathematical calculations on numeric types.
+ (Addition / String concatenation)- (Subtraction)* (Multiplication)/ (Division)% (Modulus - returns the remainder)

  Unary Operators: Require only a single operand to increment, decrement, or invert a value.
  ++ (Increment by 1, available as prefix or postfix)
  -- (Decrement by 1, available as prefix or postfix)
  ! (Logical NOT - inverts a boolean)
  - (Unary minus - negates a value)
 
    Assignment Operators: Used to assign right-hand values to left-hand variables.
    = (Simple assignment)+=, -=, *=, /=, %= (Compound assignments that compute and assign in one step

    Relational Operators: Compare two values and always return a boolean result (true or false).
    == (Equal to)
    != (Not equal to)
    > (Greater than)
    < (Less than)
     >= (Greater than or equal to)
     ><= (Less than or equal to)

    Logical Operators: Combine multiple boolean conditions.
     They support short-circuit evaluation.
    && (Logical AND - returns true if both expressions are true)
    || (Logical OR - returns true if at least one expression is true)

    Bitwise and Shift Operators:
    Manipulate the binary bits of an integer value.
    & (Bitwise AND),
    | (Bitwise OR),
    ^ (Bitwise XOR),
     ~ (Bitwise complement)
    << (Left shift),
    >> (Signed right shift),
   >>> (Unsigned right shift)
